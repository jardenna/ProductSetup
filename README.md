# React + TypeScript + Vite

Minimal React + TypeScript setup powered by **Vite** with:

- ESLint (Flat config)
- Accessibility linting (**jsx-a11y**)
- Import sorting and consistent formatting
- Stylelint for **SCSS**
- Type checking

## Requirements

- Node.js (LTS recommended)
- npm

## Getting started

Install dependencies:

```bash
npm install
```
