# React + TypeScript + Vite

Minimal React + TypeScript project powered by **Vite** with:

- ESLint (Flat config)
- Accessibility linting (**eslint-plugin-jsx-a11y**)
- Prettier formatting
- Stylelint for **SCSS**
- Type checking (TypeScript)

## Requirements

- Node.js (LTS recommended)
- npm

## Install

```bash
npm install
```
