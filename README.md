# React + TypeScript + Vite

Minimal React + TypeScript project powered by **Vite**.

## What’s included

- React + TypeScript
- Vite dev server + production build
- ESLint (Flat config)
- Accessibility linting (`eslint-plugin-jsx-a11y`)
- Prettier formatting
- Stylelint for SCSS
- Type checking (TypeScript)

---

## Requirements

- Node.js (LTS recommended)
- npm

---

## Setup

Install dependencies:

```bash
npm install
```
